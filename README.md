 #  Real Estate Frontend Website Using React js

This is a responsive real estate frontend built with Vite + React, styled using Tailwind CSS, and enhanced with Framer Motion animations. Featuring both dark & light mode, it includes pages like Home, About, Properties, Contact Us (with Gmail contact integration), and Login/Signup UI‚Äîwithout using react-router-dom.

###  Installation & Setup:  
git clone https://github.com/mohadkaif122344/Real-Estate-Website-in-react-js.git

cd Real-Estate-Website-in-react-js
npm install
npm run dev


Then open your browser and go to http://localhost:5173.

#### To build for production:

npm run build
